# Dotfiles

These are the dotfiles I use to customize my system

1. **Operating System**: Ubuntu 20.04
2. **Window manager**: Gnome
3. **Shell**: Zsh
4. **Terminal**: Guake
5. **Editor**: Neovim. CLion/PyCharm with IdeaVim when required.
6. **Browser**: Mozilla Firefox
